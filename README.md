🌌 The Void
A Digital Ritual for Mental De-cluttering By ClintonAzabuike-Creator

The Void is a high-tech philosophical tool. It is built for a single purpose: The destruction of digital and mental weight. ## 🧠 The Unthinkable Logic
Most apps are designed to capture and save. The Void is designed to release and erase.

Zero Persistence: There is no "Save" button. There is no database. Once the particle animation finishes, the data is gone from the browser's memory forever.

Canvas Engineering: Uses HTML5 Canvas API to transform your text into a physical simulation of disappearing matter.

Privacy by Design: Even the person holding the phone cannot see what was written once it is released.
